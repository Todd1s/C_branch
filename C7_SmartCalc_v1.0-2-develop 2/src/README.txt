Smart Calculator
О программе
Калькулятор реализован на графической библиотеке QT. 
При запуске программы открывается окно калькулятора, есть возможность построения графика, для этого:
Введите график функции, укажите диапазон отображаемой области определения и область значения и нажмите кнопку график
Чтобы начать пользоваться калькулятором, его необходимо установить с помощью команды make install. 
Эта команда установит программу в папку build
Чтобы удалить программу, нужно воспользоваться командой make uninstall.
